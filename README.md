# mimanjh

Personal portfolio site for Jacob Hunsaker — a Full Stack Developer.

Built with [Next.js](https://nextjs.org/) (App Router), TypeScript, Tailwind CSS, [DaisyUI](https://daisyui.com/), [Swiper](https://swiperjs.com/), and [Framer Motion](https://www.framer.com/motion/).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — build for production
- `npm run start` — run the production build
- `npm run lint` — lint the project
